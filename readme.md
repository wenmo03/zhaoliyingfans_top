## 配置
1.创建mysql数据库
```mysql
CREATE DATABASE `fans_db` CHARACTER SET 'utf8mb4' COLLATE 'utf8mb4_general_ci';
```
2.安装python扩展
```
pip install -r requirements.txt
```
3.运行测试项目
```
python manage.py runserver
```
